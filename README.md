# 📚 Roua Study

A comprehensive school study platform designed to help students succeed in their academic journey.

## 🚀 Features

- **Home Page**: Welcome and platform overview
- **Courses**: Organized courses for Collège and Lycée students
- **Study Fiches**: Downloadable study materials for Math, Physics-Chemistry, and Biology
- **About**: Learn more about Roua Study
- **Contact Form**: Get in touch with us

## 📋 Subjects Covered

- 📐 **Mathematics**
- ⚛️ **Physics & Chemistry**
- 🧬 **Biology & SVT (Natural Sciences)**

## 🛠️ Tech Stack

- **Frontend**: React 18
- **Routing**: React Router v6
- **Styling**: CSS3
- **Package Manager**: npm

## 📦 Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Steps

1. Clone the repository:
```bash
git clone https://github.com/ajjelroua-del/roua-study.git
cd roua-study
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
